Concept of WWW

  • It stands for World Wide Web.
  • It is basically a system of internet serves that support specially formatted documents.
  • It is a collection of text pages,videos,digital photograph and animatons we can access over the internet.
  • User access the World Wide Web browser.

HTTP Protocol

  1. HTTP stands for Hypertext Transfer Protocol
  2. HTTP is based on the client server architechture model and a stateless request/response Protocol
  3. 404 File Not Found is a common HTTP status code.
